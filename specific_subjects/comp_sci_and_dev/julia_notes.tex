\noindent\texttt{using Foo} imports the entire namespace, unlike in Python. However, the very Pythonic \texttt{import Foo as Bar} is now possible in Julia. This is mostly useful when the package name conflicts with an identifier; abbreviations were already possible thanks to the keyword \textt{const}.
